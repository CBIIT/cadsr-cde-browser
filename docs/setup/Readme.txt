
===============================================================================
			CDBrowser 2.1
===============================================================================

This distribution contains the application ear and the complete source for 2.1
version of CDEBrowser


The zip/gzip file containig the application ear file also contains the installation
instaructions to install CDEBrowser 2.1 on stanalone OC4J and 9ias.

The zip/gzip file containing the application source has the following directory 
structure 
	
/lib            - Third party libraries.
/config         -  Property and xml files that should be in the application 
		  classpath.
/ui		- Contains jsp,html,javascripts and images.
