==============================================================================
			CDBrowser 3.0
==============================================================================

This distribution contains application ear file, libraries, documentation, and source 
code for CDEBrowser 3.0 Seperate downloads are available for Unix and Windows platforms.

	CDEBrowser_Windows_3.0.zip
	CDEBrowser_Unix_3.0.tar.gz
	CDEBrowser_Windows_src_3.0.zip
   	CDEBrowser_Unix_src_3.0.tar.gz

CDEBrowser_Windows_3.0.zip and CDEBrowser_Unix_3.0.tar.gz contains the application
ear file for windows and unix platforms respectively along with instructions to deploy the application to
JBOSS application server.

CDEBrowser_Windows_src_3.0.zip and  CDEBrowser_Unix_src_3.0.tar.gz contains the complete
source code, descriptors and configurations files for Windows and Unix platforms respectively.
Also contained in this distribution is the readme explaining its contents.
